## Features

### Leveling System
- XP from messages and voice activity
- Auto role rewards at milestones
- Rank cards with progress tracking
- [Live leaderboard](https://trippixn.com/syria)

### TempVoice
- Create private voice channels on-demand
- Control panel: rename, lock, limit, kick/ban users
- Auto-cleanup when empty

### Media Tools
- `/download` - Download videos from social platforms
- `/convert` - Convert images/videos (format, compress, resize)
- `/translate` - Translate text via DeepL/Google
- `/image` - Google image search

### Community
- `/confess` - Anonymous confessions with mod review
- `/afk` - AFK status with return notifications
- `/quote` - Generate quote images from messages
- `/weather` - Weather lookup by city

### Fun
- Action commands (hug, slap, poke, etc.)
- Auto-reactions and responses
- Gallery channels with reactions

## Commands

| Command | Description |
|---------|-------------|
| `/rank` | View your level and XP |
| `/download <url>` | Download video from URL |
| `/convert` | Convert attached media |
| `/translate [text]` | Translate text or reply |
| `/image <query>` | Search for images |
| `/confess` | Submit anonymous confession |
| `/afk [reason]` | Set AFK status |
| `/weather <city>` | Get weather info |
| `/get avatar/banner` | Get user's avatar or banner |

## Tech Stack

- **Runtime:** Python 3.12
- **Framework:** discord.py 2.7+
- **Database:** SQLite with WAL mode
- **APIs:** DeepL, OpenWeather, Google CSE, Cobalt
